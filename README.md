# 🌍 Global GHG Emissions — Causal Tree

I started with a simple question - can I get a high-over view of the total carbon emissions globally with some level of confidence? This is the result of that question.

Here is a simple and interactive breakdown of global greenhouse gas emissions constituting 100% of ~59 GtCO₂e (2019 baseline).

**[→ Click here to access it](https://sushds.github.io/global-emissions-tree/)**

## Data sources
- IPCC AR6 Working Group III (2022)
- Climate Watch / WRI CAIT Database (2019)
- IEA World Energy Outlook 2023
- FAO FAOSTAT Agrifood Emissions
- Global Carbon Project 2023
- IMO Fourth GHG Study (2020)

## Features
- Click any node to expand subsectors
- Confidence ratings (High / Med / Low) per source
- Data stop indicators where global reconciliation breaks down
- Full source references inline
